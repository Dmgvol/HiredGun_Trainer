# HiredGun_Trainer
 Necromunda: HiredGun Trainer</br>
 Compatible with the release version (Steam and GOG).
 
 ![](HiredGunTrainer/Images/trainer.png)
 
## Controls
- F3: Noclip
- F4: Game Speed
- F5: Save Position
- F6: Teleport to saved position
- F7: Teleport to manual position
